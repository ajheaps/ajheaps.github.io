Download/Install¶

Is cf-plot already installed?¶

Jasmin@CEDA - cf-plot is available inside the Jasmin Analysis Platform - type python2.7 to start python.

Archer Jasmin RDF - jasmin.rdf.ac.uk - This has the Jasmin Analysis Platform installed - type python2.7 to start python.

Archer - type
module swap PrgEnv-cray PrgEnv-intel
module load anaconda cf
and then python to use cf-plot.

Department of Meteorology, University of Reading - The latest version of cf-plot is installed on the Reading University Meteorology department servers. Type setup canopy to set-up Python.



To install cf-plot¶

Use pip: Install cf-python and cf-plot

pip install cf-python
pip install cfplot

GitHub:

git clone git://github.com/ajheaps/cf-plot.git
(if behind a firewall such as that at the Met Office you may need to use
git clone https://github.com/ajheaps/cf-plot.git)

cd cfplot
python setup.py install

You will need to download and install cf-python to use cf-plot. This can also be done with

Other cf-plot dependencies are: Numpy, Matplotlib, Netcdf4 and Basemap.
