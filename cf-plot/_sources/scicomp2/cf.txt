Examples of data analysis and visualisation
===========================================


1) `Read in netCDF file, select a field and save data as a netCDF <http://ajheaps.github.io/cf-plot/scicomp2/cf_example1.html>`_
2) `Add a missing CF attribute and change units  <http://ajheaps.github.io/cf-plot/scicomp2/cf_example2.html>`_
3) `Data slicing and meaning <http://ajheaps.github.io/cf-plot/scicomp2/cf_example3.html>`_
4) `Cylindrical projection and polar contour plots <http://ajheaps.github.io/cf-plot/scicomp2/cf_example4.html>`_
5) `Zonal mean plots with linear and log pressure axes <http://ajheaps.github.io/cf-plot/scicomp2/cf_example5.html>`_
6) `Take two data sets and interpolate them to the same grid  <http://ajheaps.github.io/cf-plot/scicomp2/cf_example6.html>`_
7) `Globally average a timeseries and plot <http://ajheaps.github.io/cf-plot/scicomp2/cf_example7.html>`_
8) `Combining data (i.e. collection of netcdf files into single entity) <http://ajheaps.github.io/cf-plot/scicomp2/cf_example8.html>`_
9) `Ensemble average data <http://ajheaps.github.io/cf-plot/scicomp2/cf_example9.html>`


Download all data files used in the examples :download:`scicomp2_data.tgz <http://www.met.reading.ac.uk/~swsheaps/scicomp2_data.tgz>` 
Use tar -zxvf scicomp2_data.tgz on the Unix command line to extract the files.
