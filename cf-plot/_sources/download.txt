To install cf-plot
==================

Use pip: 
Install cf-python and cf-plot

::

   pip install cf-python
   pip install cfplot


GitHub:


::

   git clone git://github.com/ajheaps/cf-plot.git
   (if behind a firewall such as that at the Met Office you may need to use
   git clone https://github.com/ajheaps/cf-plot.git)
 
   cd cfplot
   python setup.py install


You will need to download and install `cf-python <http://cfpython.bitbucket.org>`_ to use cf-plot.  This can also be done with 



Other cf-plot dependencies are: Numpy, Matplotlib, Netcdf4 and Basemap.


Download the `sample netCDF datasets <http://climate.ncas.ac.uk/~swsheaps/cfplot_sphinx/data>`_
