cf_var_name¶

cfplot.cf_var_name(field=None, dim=None)[source]¶
cf_var_name - return the name from a supplied dimension
in the following order
ncvar
short_name
long_name
standard_name

field=None - field
dim=None - dimension required - ‘dim0’, ‘dim1’ etc.





Returns:name
