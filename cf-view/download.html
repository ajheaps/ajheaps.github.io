Download/Install¶

To install cf-view¶

cf-view is available as a GitHub repository. To download and install cf-view:

git clone git://github.com/ajheaps/CF-View.git

You will need to install cf-python and cf-plot to use cf-view. You can install these using pip:

pip install cf-python
pip install cf-plot

Other cfview dependencies are: gtk, Numpy, Matplotlib, Netcdf4 and Basemap.

To run cf-view¶

cd CF-View
cfview <optional netCDF, Met Office PP or fields file(s)>

The cfview script runs python2.7 as this is the version of Python in the Jasmin Application Platform (JAP) that has gtk installed. You may need to change this in the cfview script to suit your local Python distribution.

Firewall issues¶

If you are behind a firewall such as that at the Met Office you may need to use the following the download cf-view.

git clone https://github.com/ajheaps/CF-View.git
